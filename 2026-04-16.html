プルーフ・オブ・ワークになった

LLMがトークン予算を無制限に投じて脆弱性を探す時代、セキュリティの優劣はもはや「賢さ」ではなく「予算」で決まる。攻撃側より多くのトークンを使って防御する——それが現代セキュリティの本質だ。開発パイプラインに継続的なAIセキュリティ監査フェーズを組み込む必要がある。

05  —  オープンソース文化
🔒 オープンソース終了宣言

Cal.com、クローズドソースに転換——
AIセキュリティ時代の苦渋の決断

5年間オープンソースを標榜してきたカレンダーSaaS「Cal.com」が本番コードベースをプライベートリポジトリへ移行。理由は「AIによる攻撃の高度化でセキュリティリスクが増大した」から。ただしコミュニティ向けに「Cal.diy」として OSS版を維持する方針。

「コードを公開すること自体がリスクになる時代が来た」——これはSaaS業界全体への警鐘として受け取られている。

  • 🔒
    本番リポジトリ非公開化 公開GitHubリポジトリから本番コードを削除。外部からの覗き見を完全遮断。
  • 🌱
    Cal.diy として OSS継続 セルフホスト・コミュニティ主導版を「Cal.diy」として維持。ホビイスト向け。
  • 🤖
    AI脅威が転換のトリガー LLM支援の自動脆弱性探索が現実的脅威となり、公開コードのリスクが急増。
  • 💭
    業界への示唆 HashiCorp・Redis・Elasticに続くOSS→商用化の波。AIセキュリティが加速。

06  —  奇妙な科学 / 生物
奇妙な生物

ウミウシが光合成する——
「海のヒツジ」と呼ばれる動物の謎

Costasiella kuroshimae（コスタシエラ・クロシマエ）は体長5〜10mmの極小ウミウシ。食べた藻類から葉緑体を盗み取り、自分の細胞に組み込んで光合成を行う。動物でありながら「太陽光で生きる」唯一の生物の一つ。その羊のような顔立ちが「海のヒツジ（Sea Sheep）」の愛称を生んだ。

5mm
成体の体長（最大10mm）
窃盗
葉緑体を藻から盗む「kleptoplasty」という機能
1993
日本・黒島沖での発見年

07  —  開発ツール / CLI

Wacli——WhatsAppを
ターミナルから完全操作

steipete氏が開発したWacliは、WhatsApp Web プロトコルを使いターミナルからメッセージ送受信・履歴検索・グループ管理などを実現するCLIツール。Go製で Homebrew インストール可能。JSON出力モードでシェルスクリプトや自動化パイプラインと統合できる。公式APIではなく「whatsmeow」ライブラリを使用。

メッセージアーカイブ、オフライン全文検索、メディア一括ダウンロードなど、公式アプリにない機能が充実。

wacli sync
メッセージ履歴をローカルに同期
wacli search "keyword"
オフラインで全履歴をフルテキスト検索
wacli send @contact "message"
コマンドラインからメッセージ送信
wacli groups list --json
グループ一覧をJSON出力でスクリプト連携
wacli media download
同期済みメディアを一括ダウンロード

08  —  システム設計 / アーキテクチャ
🗄️ システム設計論

あなたのアプリ、
データベースは本当に必要ですか？

RAM 収まるなら DB 不要
1フィールド 検索ならハッシュマップで十分
224 HNコメント数（活発な議論中）

ほとんどのアプリは自分で思うより小さい。データがRAMに収まりIDルックアップが主な操作なら、インメモリハッシュマップはDBより高速。「DBのファイルを使うか自分のファイルを使うかの違いだ」——この主張が224コメントの活発な議論を生んでいる。

  • ✅
    DBが必要な時 複数フィールドにまたがる複雑なクエリ・JOIN・複数エンティティの原子的トランザクションが必要な場合。
  • 🗂️
    フラットファイルで十分な時 シングルIDルックアップ・RAM収容可能サイズ・初期段階のプロダクト。速度と単純さを優先。
  • 🔀
    DBに移行するタイミング 並行書き込みが増加・多フィールド検索が頻発・データサイズがメモリを超えた時。

Anna's Archive、Spotifyの
$3.22億訴訟に無抵抗敗訴

世界最大の書籍海賊サイト「Anna's Archive」がSpotifyから提起された3億2,200万ドルの著作権侵害訴訟に応答せず、デフォルト判決（欠席裁判）で敗訴。訴訟には登場しなかった同サイトの実態、Sci-Hubとの関係、そしてこの結果が海賊行為コミュニティに与える影響を308コメントが論じている。

📚
Anna's Archive とは Library Genesis・Sci-Hub・Z-Library などの複数海賊サイトを横断検索できるメタ検索エンジン。数百万冊の学術書・論文・一般書籍を無料提供。欧米の学術研究者にとって「払えない論文アクセス」の代替として広く利用されていた。
