ENGINEERING — 浮動小数点の真実

浮動小数点の等値比較は
実はOKなケースが多い——常識を疑え

「浮動小数点同士を == で比較してはいけない」という格言を覆す記事が164ポイント・112コメントで注目されている。著者のlisyarusは「それは常に誤りではなく、文脈次第で完全に正しい」と主張する。具体的には①同一計算経路を経た値の比較、②定数との比較、③フラグ的用途の浮動小数点——などでは == 比較が適切であり、epsilon 比較の方が逆に問題を起こす場合もあるという。実践的な例とその数学的根拠を丁寧に説明する良記事。
✓ OK — 定数との比較
if x == 0.0 { ... }
初期化確認・番兵値用途では == で全く問題なし
✓ OK — 同一計算結果
a == b // 同じ式から
同一計算経路を経た値は同一ビット列を持つ
△ 注意 — 異なる計算経路
sqrt(x)*sqrt(x) == x
このケースこそ epsilon 比較が必要な典型例
